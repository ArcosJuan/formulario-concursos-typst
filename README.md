# Formulario de inscripción a concurso regular o a selección interina de docentes auxiliares.
Réplica en Typst del formulario homónimo en https://www.dc.uba.ar/descargas/

## Como completar
1. Rellenar con tu información los archivos: (Detalles del cómo más abajo)
  - `antecedentes_docentes.typ`
  - `antecedentes_cientificos.typ`
  - `antecedentes_extension.typ`
  - `antecedentes_profesionales.typ`
  - `titulos_otros.typ`
  - `calificaciones.typ`
  - `info_licencias.typ`
2. Reemplazar el archivo `firma.jpg` por tu firma. Luego, en el archivo `concurso.typ` cambiar en donde pone firma.jpg por el nombre del archivo con tu firma.

## Genera tu documento!
Ejecuta el comando `make` desde dentro de esta la carpeta.
